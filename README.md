# 🌸 Bloomore

## 📖 About the Project
The Flower Store is a web-based e-commerce application that allows users to browse and purchase flowers and seeds online.

## ✨ Features
- User Registration & Login
- Browse Flowers and Seeds
- Search for Products
- Shopping Cart
- Product Details
- Responsive User Interface
- Database Integration with MySQL

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- PHP
- MySQL

## 🚀 How to Run
1. Install XAMPP.
2. Copy the project into htdocs.
3. Start Apache and MySQL.
4. Import the database.
5. Open:
   http://localhost/flowers_store
